5. positions

position:
1. relative = positioned relative to where it normally
2. fixed = positioned relative to the viewport
3. absolute = positioned relative to the nearest ancestor
4. sticky = positioned based on scroll position (almost like fixed but have margin with other element)
5. static = default for an element
