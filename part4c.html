3. Display Property

display property speficies how a element displayed
block-level = start on an new line, take up the full width available
(h1, div, p, form, header, footer)
inline = do not start on new line, width is limited what is need(take room as necessary)
(span, a, img)

display:block -- affect span
display:inline -- affect div
display:inline-block
display:none
visibility:hidden
visibility:visible
